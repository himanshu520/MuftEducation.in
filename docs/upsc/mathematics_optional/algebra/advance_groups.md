## Theorem AG.1:

$U(m)$ is cyclic precisely if $m \in \\{2, 4, p^r, 2p^r\\}, \text{ where } p \neq 2$ and is prime. 

**Proof:** 

Case 1: $m = 2^r; r > 2$

Let $\lambda = 2^{r - 1} - 1, \beta = 2^r - 1$

$\lambda^2 = 2^{2r - 2} + 1 - 2^r = 2^{r - 2} + 1$
